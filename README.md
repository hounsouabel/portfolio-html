# Portfolio HTML

Ce projet est un portfolio personnel conçu pour présenter mes compétences, mes projets et mon parcours professionnel. Il est entièrement développé en HTML, CSS et JavaScript.

## Fonctionnalités
- Navigation fluide entre les sections avec mise à jour de l'URL.
- Présentation des compétences avec des barres de progression.
- Affichage des projets avec des modales pour plus de détails.
- Téléchargement du CV.

## Hébergement
Ce site est hébergé sur GitHub Pages. Vous pouvez le consulter à l'adresse suivante :
[https://<votre-nom-utilisateur>.github.io/<nom-du-dépôt>](https://<votre-nom-utilisateur>.github.io/<nom-du-dépôt>)

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Owl Carousel

## Installation
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/hounsouabel/portfolio-html.git
   ```
2. Ouvrez le fichier `index.html` dans votre navigateur.
